# Gudang Link — Social Link Hub 🔗

> **Linktree Alternative:** A lightweight, self-hosted social link hub built with Vite + vanilla JavaScript.

---

## ✨ Features

- **🔗 Multiple Links** — Display all your social media & important links in one page
- **🎨 Customizable** — Clean, minimal design with CSS styling
- **⚡ Lightning Fast** — Built with Vite for instant hot-reload during development
- **📱 Responsive** — Works on desktop and mobile

---

## 🏗️ Tech Stack

| Layer | Technology |
|-------|------------|
| **Bundler** | Vite |
| **Frontend** | Vanilla JavaScript, HTML5, CSS3 |
| **Deployment** | Static hosting (Netlify, Vercel, GitHub Pages) |

---

## 🚀 Getting Started

```bash
# Clone
git clone https://github.com/bakadenir/gudang-link.git
cd gudang-link

# Install
npm install

# Run development server
npm run dev

# Build for production
npm run build
```
